# ERP 系统产品介绍（云原生版）

## 概述

云原生 ERP 套件，覆盖财务、供应链、生产、人力资源、客户关系五大模块。基于 Kubernetes 部署，微服务架构，可独立扩展。

## 模块清单

- **财务管理**：总账、应收应付、固定资产、税务、成本核算
- **供应链**：采购、库存、销售、物流、供应商门户
- **生产管理**：MRP、MES 集成、工序管理、质量管理
- **人力资源**：组织架构、薪酬、绩效、招聘、培训
- **客户关系**：线索、商机、合同、客服、工单

## 部署方式

- **SaaS 订阅**：按用户数 / 月付费，含 99.9% SLA
- **私有化**：部署在客户机房或专有云，含数据隔离
- **混合云**：核心数据私有，AI 能力调用云端

## 集成能力

- RESTful API + GraphQL
- 预置 50+ 行业连接器（SAP、Oracle、Salesforce、钉钉、企业微信）
- 事件总线（Kafka）支持实时数据流

## 适用规模

- 中型：100-500 人
- 大型：500-5000 人
- 集团：5000+ 人，多组织多账套
